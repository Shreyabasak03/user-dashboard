# User Dashboard

## Overview
A simple React user dashboard that fetches users from https://jsonplaceholder.typicode.com/users and shows them in a table.

## Features
- Fetch users from API
- Display Name, Phone, Company, Image, Coins
- Responsive table

## How to Run
1. Install dependencies:
   npm install

2. Start development server:
   npm start

The app will open at: http://localhost:3000

## Deployment (Vercel)
1. Login to https://vercel.com with GitHub
2. Click “New Project”
3. Select your GitHub repo (user-dashboard)
4. Build Command: npm run build
5. Output Folder: build
6. Deploy → Get live link

## Tech Stack
- React (CRA)
- CSS
- Fetch API
